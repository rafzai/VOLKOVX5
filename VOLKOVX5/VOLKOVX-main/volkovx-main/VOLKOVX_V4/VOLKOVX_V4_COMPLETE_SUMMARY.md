## 📈 MATHEMATICAL VALIDATION

### KL Divergence (Bregman)
```
D_KL(p || q) = Σ p_i * ln(p_i / q_i)

Example:
  Prices: YES=$0.62, NO=$0.33 → Sum=$0.95
  Fair: μ*_YES=0.6526, μ*_NO=0.3474
  Profit = -ln(0.95) = 5.13% per $1
```

### Frank-Wolfe Convergence
```
Theorem: Converges to ε-optimal in O(1/ε) iterations
Typical: 50-150 iterations for prediction markets
vs: 2^63 brute force → IMPOSSIBLE
```

### Kelly Criterion
```
Conservative (1/4 Kelly):
  f = 0.25 * kelly_fraction
  Size = 0.25 * portfolio * kelly
  Constraints: 5% max portfolio, 30% max order book
```
